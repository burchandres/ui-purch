# Purch UI

Web app for Purch service

## Stack
- Language: Typescript
- Framework: React
- Build tool: Vite
- Router: TanStack
- Components: shadcn

## Get started
1. [install pnpm](https://pnpm.io/installation) globally
1. install node dependencies: `pnpm i`
1. start up app: `pnpm dev`

See other available commands: `pnpm run`
